# 🐾 돈땃쥐미 (FITCOIN)

> **데일리 운동 미션으로 캐릭터를 키우고, 실물 기프티콘을 받자!**

>
> ![돈땃쥐미 캐릭터들 운동 모습](Frontend/fitCoin/public/splash.gif)

<br>

## 서비스 소개

**돈땃쥐미**는 하루하루 운동 미션을 수행하며 귀여운 캐릭터를 성장시키는 **앱테크 PWA 서비스**입니다.

미션을 완료하면 포인트가 쌓이고, 캐릭터가 10경험치를 달성해 **졸업**하면 실물 기프티콘을 보상으로 받을 수 있어요.  
방 꾸미기, 뽑기, 도감 수집 등 다양한 게임 요소로 운동 습관 형성을 재미있게 도와줍니다.

<br>

## 주요 기능

| 기능                | 설명                                               |
| ------------------- | -------------------------------------------------- |
| 🏋️ **데일리 미션**  | 매일 미션 3개 제공, 수행 시 포인트 적립            |
| 🐹 **캐릭터 성장**  | 미션 완료 → 경험치 획득 → 단계별 진화              |
| 🎓 **캐릭터 졸업**  | 10경험치 달성 시 졸업 + 기프티콘 보상              |
| 🛋️ **방 꾸미기**    | 가구 배치·테마 완성으로 나만의 공간 꾸미기         |
| 🛒 **상점 / 뽑기**  | 포인트·코인으로 가구, 캐릭터 변경권, 기프티콘 뽑기 |
| 💱 **환전소**       | 포인트 → 코인 환전 (매일 변동 환율 적용)           |
| 📅 **스트릭**       | 연속 출석일 추적 및 월별 달력 제공                 |
| 📖 **도감**         | 캐릭터·가구 수집 도감 (미수집 시 실루엣)           |
| 🤸 **AI 운동 분석** | MediaPipe 기반 실시간 자세 피드백                  |
