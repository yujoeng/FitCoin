## 📌 PR 유형

어떤 변경 사항인지 해당하는 항목에 체크해주세요.

- [ ] 새로운 기능 추가 (feature)
- [ ] 버그 수정 (bugfix)
- [ ] 코드 리팩토링 (refactor)
- [ ] 의존성 업데이트 (dependency)
- [ ] 문서 수정 (docs)
- [ ] 테스트 추가/수정 (test)
- [ ] 기타 (기타 내용을 아래에 작성해주세요)

## ✅ 셀프 체크리스트

PR을 올리기 전에 아래 항목을 확인해주세요.

- [ ] 로컬에서 빌드가 정상적으로 완료되었다
- [ ] 새로 추가한 코드에 적절한 테스트를 작성했다
- [ ] 기존 테스트가 모두 통과한다
- [ ] 불필요한 `console.log`, 주석, 디버그 코드를 제거했다
- [ ] PR 제목이 변경 사항을 명확히 나타낸다
- [ ] 리뷰어에게 전달할 컨텍스트가 충분히 기술되어 있다
