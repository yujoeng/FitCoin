### 스트릭 데이터

- 발표 당일(2026년 3월 31일) 기준으로, 캐릭터 레벨이 9가 되어있고 당일의 미션은 수행하지 않은 스트릭 상태를 만든다.

```bash
-- 시연 날짜 기준: 2026-03-31
-- user_id = 32
-- 캐릭터 입양일: 2026-03-15, last_updated_date=2026-03-14 (재계산 트리거용)
-- 3/15~3/30 패턴으로 자동 계산 시 exp=9 도달

-- ─────────────────────────────────────────────
-- 1. user_character 설정
-- last_updated_date = 3/14 (입양 전날) → 조회 시 3/15~3/30 재계산
-- ─────────────────────────────────────────────
UPDATE user_character
SET
    exp               = 0,
    status            = 'GROWING',
    adoption_date     = '2026-03-15 00:00:00',
    last_updated_date = '2026-03-14'
WHERE user_id = 32 AND status != 'GRADUATED';

-- ─────────────────────────────────────────────
-- 2. streak 데이터 초기화
-- ─────────────────────────────────────────────
DELETE FROM streak WHERE user_id = 32;

-- ─────────────────────────────────────────────
-- 2025년 12월 (26일 출석)
-- 출석: 1~5, 8~14, 16~22, 24~28, 30~31
-- 미출석: 6, 7, 15, 23, 29
-- data = 1874837407
-- ─────────────────────────────────────────────
INSERT INTO streak (user_id, year_and_month, data)
VALUES (32, '2025-12-01', 1874837407);

-- ─────────────────────────────────────────────
-- 2026년 1월 (27일 출석)
-- 출석: 2~7, 9~15, 17~23, 25~29, 31
-- 미출석: 1, 8, 16, 24, 30
-- data = 1602191230
-- ─────────────────────────────────────────────
INSERT INTO streak (user_id, year_and_month, data)
VALUES (32, '2026-01-01', 1602191230);

-- ─────────────────────────────────────────────
-- 2026년 2월 (25일 출석)
-- 출석: 1~4, 6~11, 13~18, 20~25, 27~28
-- 미출석: 5, 12, 19, 26
-- data = 234616815
-- ─────────────────────────────────────────────
INSERT INTO streak (user_id, year_and_month, data)
VALUES (32, '2026-02-01', 234616815);

-- ─────────────────────────────────────────────
-- 2026년 3월 (23일 출석, 3/31 미출석)
-- 출석: 1~3, 5~10, 12~14, 18~24, 27~30
-- 미출석: 4, 11, 15, 16, 17, 25, 26, 31
--
-- 3/15~3/30 경험치 시뮬레이션 (10 미도달):
-- 15~17: 미출석 → exp=0 (floor)
-- 18~24: 출석 7일 → exp=7
-- 25~26: 미출석 2일 → exp=5
-- 27~30: 출석 4일 → exp=9 ✓ (최대 9, 10 미도달)
--
-- bit 계산:
-- 1~3:   (1<<3)-1                    = 7
-- 5~10:  (1<<10)-(1<<4)              = 1008
-- 12~14: (1<<14)-(1<<11)             = 14336
-- 20~28: (1<<28)-(1<<19)             = 267911168
-- data = 7|1008|14336|267911168 = 268435448
    -- ─────────────────────────────────────────────
# DELETE FROM streak WHERE user_id = 20 AND year_and_month = '2026-03-01';
INSERT INTO streak (user_id, year_and_month, data)
VALUES (32, '2026-03-01', 535822336);

-- ─────────────────────────────────────────────
-- 확인 쿼리
-- ─────────────────────────────────────────────

SELECT id, user_id, exp, status, adoption_date, last_updated_date
FROM user_character
WHERE user_id = 32;
-- 기대값: exp=0 (조회 시 자동 재계산 → 9), last_updated_date=2026-03-14

commit;
```

## 캐릭터 도감

- 17가지 캐릭터 중, 5가지 캐릭터를 unlock 상태로 둔다.
- 이를 위하여, unlock로 둘 캐릭터를 user_character 테이블에 GRADUATED인 행들을 추가한다.

```bash
-- user_id = 32, 랜덤 선택 12개 캐릭터 GRADUATED 상태로 unlock
INSERT INTO user_character (adoption_date, exp, graduation_date, status, character_id, user_id, last_updated_date)
VALUES
  (NOW(), 0, NOW(), 'GRADUATED', 1,  32, CURDATE()),  -- 강아지
  (NOW(), 0, NOW(), 'GRADUATED', 2,  32, CURDATE()),  -- 개구리
  (NOW(), 0, NOW(), 'GRADUATED', 7,  32, CURDATE()),  -- 드래곤
  (NOW(), 0, NOW(), 'GRADUATED', 15, 32, CURDATE()),  -- 토끼
  (NOW(), 0, NOW(), 'GRADUATED', 17, 32, CURDATE());  -- 호랑이
  commit;
```

# 시연 시나리오

### 시연 환경
- 시연 날짜: 2026-03-31
- 시연 계정: `user_id = 32`
- 사전 세팅: 시연용 DB 데이터 삽입 완료

### 시연 순서

#### 1. 홈 화면
- 접속 시 캐릭터, 방 꾸미기, 스트릭 바, 보유 자산(포인트/코인) 확인
- 스트릭 바: 최근 7일 출석 현황 표시
- 우상단 설정 버튼으로 설정 페이지 이동 가능

#### 2. 미션 수행
- 하단 탭바 → 홈 화면 → 미션 버튼 클릭
- 미션 카드에서 운동 종류 확인 후 **바로 시작** 클릭
- 운동 수행 (MediaPipe 자세 인식)
- 완료 시 포인트 획득 및 스트릭 체크, 경험치 반영

#### 3. 스트릭 / 마이페이지
- 하단 탭바 → 마이페이지
- 월별 출석 달력 확인
- 연속 스트릭 일수 확인

#### 4. 캐릭터
- 하단 탭바 → 캐릭터
- 현재 경험치(9/10) 확인
- 미션 완료 후 경험치 10 도달 → **졸업하기** 버튼 활성화
- 졸업 시 기프티콘 지급 확인

#### 5. 환전소
- 하단 탭바 → 지갑 → 환전소
- 현재 환율 확인
- 환율 변동 그래프 (기간 필터: 전체/1년/3개월/1개월/1주)
- 포인트 입력 후 **환전하기** → 환전 성공 모달 확인

#### 6. 방 꾸미기
- 홈 화면 → 방 꾸미기 버튼
- 보유 가구 목록에서 슬롯별 가구 선택
- 저장 후 홈 화면에서 변경된 방 확인

#### 7. 도감
- 하단 탭바 → 캐릭터 → 도감
- 졸업시킨 캐릭터 목록 확인
- 캐릭터 클릭 → 상세 이미지(졸업 전/운동 중/졸업 후) 슬라이드 확인

#### 8. 광고 시청
- 홈 화면 → 광고 버튼 클릭
- 광고 시청 완료 시 포인트 획득 확인
